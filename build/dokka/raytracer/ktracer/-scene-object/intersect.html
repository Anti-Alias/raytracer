intersect

abstract fun intersect(rayOrigin: Vec3, rayDirection: Vec3): Intersection?

Attempts to solves for an intersection between a ray and this SceneObject.

Parameters

rayOrigin - Origin of the ray being casted.

rayDirection - Direction of the ray being casted.

Return
Intersection that was found, or null when there was no intersection. No intersection should be returned when this object is behind the ray. For instance, if t < 0.
